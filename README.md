# Deep Past Challenge
A competition project to develop machine translation model from akkadian to english
Machine Translation project developed for [Deep Past Challenge - Translate Akkadian to English](https://www.kaggle.com/competitions/deep-past-initiative-machine-translation). The goal is to fine tune and/or create custom large language model capable of easiliy translating dead Akkadian language to english. The project is ongoing until around 1st april and may receive changes. Current solution relies on fine tuned Google [T5 model](https://huggingface.co/docs/transformers/model_doc/t5).
